
This toolbox provides a periodicity estimation algorithm developed in the paper:
Lin Zhang and Petko Bogdanov,
"Period Estimation For Incomplete Time Series. " (DSAA 2020)


To execute, run the demo.m file.
     
	 
     
If you are using our code for your research please cite:
"Period Estimation For Incomplete Time Series.", 
Lin Zhang and Petko Bogdanov, DSAA2020
